# Data

This project uses the IEEE-CIS Fraud Detection dataset from Kaggle.

Due to licensing and size constraints, raw data files are not included in this repository.

Expected structure:

data/raw/
- train_transaction.csv
- train_identity.csv
- test_transaction.csv
- test_identity.csv

Download instructions:
1. Create a Kaggle account
2. Join the IEEE-CIS Fraud Detection competition
3. Download and unzip the data into data/raw/
